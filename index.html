自我介绍

一名对游戏开发充满热情的应届生。在2025年CIGA Game Jam中，与团队在48小时内开发的《活全家博物馆》荣获福州站一等奖，我主要负责UI系统实现与音效集成，虽然过程中遇到许多挑战，但此次经历极大锻炼了我在高压环境下的快速学习与执行能力。

个人也在探索游戏开发的其他各个方面，实现一些小型原型，并善于利用Yarn Spinner等工具高效实现设计目标。我享受游戏从设计到落地的过程，并乐于通过系统性拆解优秀游戏来深度学习设计理念，不断提升开发水平。

期待能为贵团队带来价值，共同打造出色的游戏体验。

我的技能

以下是目前掌握熟悉的技术

  • Unity 开发
  • C#编程
  • UI设计
  • Git版本控制
  • 音效设计
  • 问题排查

通过这两个项目，不仅锻炼了技术执行能力，还懂得了如何定位问题所在，与团队协作能力，并将设计需求转化为流畅的用户体验。

我的项目

我的开发重心围绕如何用代码更好的支撑游戏的玩法和叙事，构建直观简洁的UI，实现有趣的游戏逻辑，目标都是为了创造一个流畅具有沉浸感的体验。

《活全家博物馆》

CIGA Game Jam 福州站一等奖作品。一款48小时内完成的幽灵主题游戏。我在团队中主要负责UI系统的代码实现与音效集成，保障了项目的完整交付。

《罪列囚途》

个人全栈开发。基于Yarn Spinner插件开发了多分支对话系统，并实现了其与游戏谜题（密码门）的联动，验证了叙事驱动游戏的核心技术可行性。
